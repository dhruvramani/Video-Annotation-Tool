How to use this?

  1. Each utterance video is to be annotated with the following labels: Funny and Sarcastic.
  2. If the utterance video is a joke or contains humor, then please select "Yes" for the Funny label, else "No".
  3. If the utterance video is a sarcastic statement made by the speaker, then please select "Yes" for the Sarcastic label, else "No".
  4. If the utterance video has a positive statement made by the speaker, then please select "Positive" for the Sentiment label, else "Negative".
  5. If you find any misalignment between the video, transcript or audio (eg. video is blank, audio or subtitle is wrong etc.) please select the "Bad Sentence" tag.
  6. Click "Submit & Next" to submit the labels for the current utterance.
